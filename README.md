# common-tools
> 开发常用工具类, 为减少重复造轮子而努力!<br />
> 包含的语言：Java, Python, Shell, JavaScript

## 开发模式
- 受保护的分支: master, release-*
- 开发分支: dev-*
- Java 基于JDK 1.8
- Python 基于Python 2.x

## 准入规则
- 文件必须带有文件描述信息
- 代码内置属性必须带有注释, 包括但不限于属性, 类, 方法
- commit msg 应针对一个文件或一个模块的多个文件
- commit msg 应尽可能的涵盖所有改动描述
